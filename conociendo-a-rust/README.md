Para ejecutar los ejemplos de C:

````
> cd src/ejemplos_c
> gcc ejemplo1.c -o ejemplo1 && ./ejemplo1
> gcc ejemplo2.c -o ejemplo2 && ./ejemplo2

````

En el caso de Rust:
````
> cd src/
> cargo run --bin ejemplo1
> cargo run --bin ejemplo2
> cargo run --bin ejemplo3
> cargo run --bin ejemplo4
````
